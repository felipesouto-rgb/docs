## O que é o Bloxs Playbook

É onde mora **como a Bloxs opera**. Toda vez que você tiver uma dúvida operacional — como vendemos, como se faz um deal, o que é um FIDC, o que pode e o que não pode — a resposta está aqui. Se não estiver, é uma falha do Playbook, não sua: avise e a gente corrige.

O Playbook não existe para explicar teoria. Existe para você **executar**. Cada página termina dizendo o que fazer e para onde ir.
