## Como usar em call

Abra esta página antes de falar com parceiro ou tomador. Cada linha tem: **o que pode dizer**, **o que te processa**, **por quê**, **quem pode usar**, **canal** e se **precisa aprovação**. Se a situação não estiver mapeada, pare e escale.

## Apresentação da Bloxs

|  |  |
| --- | --- |
| ✅ Pode dizer | "A Bloxs é a infraestrutura de mercado de capitais que estrutura operações de crédito para empresas do middle-market." |
| ❌ Não pode | "A Bloxs é uma corretora onde você investe e tem retorno garantido." |
| Por quê | Segundo caracteriza varejo \+ promessa. A Bloxs não é corretora e não promete retorno. |
| Quem pode | Todos os perfis |
| Canal | Qualquer |
| Aprovação | Não (institucional aprovado) |

## Conversa com parceiro Sell Side

|  |  |
| --- | --- |
| ✅ Pode dizer | "Você origina a oportunidade e a Bloxs cuida da estruturação e distribuição pelas entidades licenciadas." |
| ❌ Não pode | "Você pode distribuir e recomendar os produtos direto pros seus contatos." |
| Por quê | Distribuir/recomendar é atividade regulada; parceiro não-licenciado não pode. |
| Quem pode | Colaborador interno, BP autorizado |
| Canal | Qualquer (sem oferta) |
| Aprovação | Não para o enquadramento; sim para material de operação |

## Por canal

| Canal | Regra |
| --- | --- |
| **WhatsApp** | Relacionamento e logística, sim. **Oferta/produto/rentabilidade, nunca.** Independent Banker: proibido qualquer conteúdo de oferta. |
| **LinkedIn** | Institucional aprovado, sim. Oferta de valor mobiliário, não. |
| **E-mail** | Apresentação institucional, sim. Material de oferta só via entidade autorizada, com peça aprovada. |

## Next action

- Situação mapeada aqui? Use a coluna "pode dizer" literalmente.
- Não mapeada, ou tocou investidor/rentabilidade/liquidez/oferta? **Pare e escale para Compliance.**
- Registre a interação.

## Red flags

- Vontade de "adiantar" condições para fechar mais rápido.
- Qualquer verbo de promessa: rende, garante, seguro, sem risco.
- Pedido para mandar material por WhatsApp a investidor.
- Independent Banker querendo falar de produto/oferta.
