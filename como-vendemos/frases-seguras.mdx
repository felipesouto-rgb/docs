## O que é isso

O comercial vive na linguagem. Uma frase errada cria passivo regulatório e reputacional mesmo com boa intenção. Este é o guia do que nunca dizer e do que dizer no lugar.

## Proibida → Segura

| Nunca diga | Diga no lugar |
| --- | --- |
| "Garantimos a captação" | "Coordenamos e colocamos junto ao buy side; o apetite é do mercado" |
| "Rende CDI \+ X, garantido" | "A remuneração é definida na estrutura e não é garantida" |
| "Tem liquidez" | "Liquidez depende do instrumento e do secundário; não prometemos" |
| "Seu deal será aprovado" | "Passa pelo Fundability Test e pela estruturação; não há aprovação automática" |
| "Somos a XP dos estruturados" (com externo) | "Somos a infraestrutura de IBaaS para o mercado de capitais privado" |
| "Pode distribuir pela sua base" | "A distribuição é atividade regulada das entidades Bloxs; você origina" |
